Estrutura condicional

Sintaxe do comando if.

if(condição){
  bloco de código executa se a condição for verdadeira;
}

Sintaxe do if-else

if(condição){
  bloco de código que executa se a condição for verdadeira;
}else{
  bloco de código que executa se a condição for falsa.
}

O comando if-else funciona se a gente não usar as chaves porem se a gente desejar executar mais de um comando devemos concatena-los usando &, o que não é uma pratica muito interessante. por isso usamos as {} e temos liberdade para escrever o código de forma livre dentro dessas chaves.

Switch case

Sintaxe do switch case

Switch(expressão){
    case (situação 1):
        comando que executa
        break (parada da execução)
    case (situação 2):
        comando que executa
        break (parada da execução)
    case (situação 3):
        comando que executa
        break (parada da execução)
    Default (padrão, resposta caso não atenda as demais opções):
        comando que executa
        break (parada da execução)
}
